# 🔮 fortune-app 개발·배포 정리

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | 오늘의 운세 |
| 기술 스택 | React 18 + Vite |
| 배포 플랫폼 | Vercel (무료) |
| GitHub | https://github.com/sunminji/fortune-app |
| 배포 URL | https://fortune-app-alpha.vercel.app |

---

## 2. 프로젝트 구조

```
src/
├── main.jsx
├── App.jsx / App.module.css
├── index.css
├── components/
│   ├── BirthInput.jsx             # 생년월일 입력 화면
│   ├── BirthInput.module.css
│   ├── FortuneResult.jsx          # 운세 결과 화면
│   ├── FortuneResult.module.css
│   ├── TarotCard.jsx              # 전체운/카테고리별 카드
│   ├── TarotCard.module.css
│   ├── TarotReading.jsx           # 타로 카드 리딩 기능
│   ├── TarotReading.module.css
│   └── StarField.jsx              # 배경 파티클
└── utils/
    ├── fortune.js                 # 운세 데이터 및 계산 로직
    ├── zodiac.js                  # 별자리/띠 계산
    └── tarot.js                   # 메이저 아르카나 22장 데이터
```

---

## 3. 주요 기능

| 기능 | 설명 |
|------|------|
| 생년월일 입력 | 직접 입력 또는 📅 버튼으로 달력 선택 |
| 전체운 카드 | 오늘의 전체 운세 (별점 + 설명 + 조언) |
| 카테고리별 운세 | 애정운 / 금전운 / 건강운 / 직업운 |
| 오늘의 행운 | 색 / 숫자 / 물건 / 시간 / 방향 |
| 타로 카드 리딩 | 7장 중 3장 선택 → 과거/현재/미래 해석 (역방향 포함) |

---
